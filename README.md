Task 6: Sales Trend Analysis Using SQL
Objective
Analyze monthly sales trends using SQL aggregations in MySQL.
The goal is to calculate monthly revenue, order volume, and identify top-performing months using time-based grouping.

Tools Used
MySQL Workbench
MySQL Server
online_sales table

Topics Covered
SELECT, WHERE, ORDER BY, GROUP BY
EXTRACT(), YEAR(), MONTH() for date analysis
Aggregate functions: SUM(), COUNT(DISTINCT)
Time-based grouping (monthly and yearly trends)
Sorting results
Limiting results using LIMIT
Handling NULL values using IFNULL()

Files Included
task6_queries.sql — all SQL queries used for revenue & trend analysis
CSV — result from MySQL Workbench
README.md — documentation explaining the task
